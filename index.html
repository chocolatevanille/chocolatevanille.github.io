My Projects

GodmasterHelper

Object Oriented Programming

This executable allows players of the hit video game Hollow Knight to track their progress and abilities in the Godmaster DLC.

Using Python's class system, I assigned information to each boss in the DLC, such as name, pantheons, and some numbers data. From there, I designed a UI that would allow for user input and managed the details of how many times they died to each boss in a way that gave players info on what they needed to work on to improve. I made sure data could be stored in files and loaded as needed. Once base functionality was implemented, I designed a full GUI in Python complete with buttons, drop-down menus, images, pop-up windows, and more, with the goal of enhancing user experience, particularly in usability. Finally, I exported the program as an executable and published it on GitHub.

Recipes

Knowledge, Representation, and Reasoning (KRR) AI in Python

This executable uses an AI ontology to provide ingredient substitutions (based around food preference and allergens) for a group of meal recipes.

I created a scalable KRR database in Python, representing shared qualities of elements via genLs; added elements (represented by isA statements) to the database and had them inherit genL qualities via backward chaining; and used forward chaining and reasoning to determine the qualities of ingredients in the recipes.

Please note: I use "I" here despite this being a group project because I did all of the work. I do not mean most of the work. I mean literally, all of it. I envisioned the premise, wrote every line of code (as evidenced by GitHub history), recorded the demo, and wrote the final project report. If the team slacks, I am willing to put in the work.
